# LewEngine
LewEngine - graphics engine based on Vulkan API

## Steps
Starting point of the project: **./blew**

## Options
- **-b/--build**:   Build the project for the specified target and method (app, test, ..)
- **-s/--setup**:   Install packages and prepare environment for all the methods
- **-t/--test**:    Run tests included or apart from the build option
- **-x/--execute**: Execute the application, UI or the current state
- **-i/--install**: Instal the current state of the project on the machine
- **-c/--clean**:   Clean the workspace from generated builds, logs, containers, etc.
